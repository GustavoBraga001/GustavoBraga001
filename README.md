Hi! 👋 

I'm a Computer Science graduate from the Federal University of Paraná (UFPR), currently living in São Paulo, Brazil. I've been working in computer graphics for almost 30 years, combining technical problem-solving with visual storytelling.

During this journey, I've worn many hats—3D Generalist, Technical Director, Pipeline Specialist, and VFX Supervisor. I spend most of my time with Maya and Houdini, lighting and rendering with the major render engines in the industry, and I enjoy automating repetitive tasks with Python whenever I get the chance.

These days, one of my main interests is building VFX pipeline tools with Python, integrating production management systems like Flow with applications such as Maya and Houdini to simplify production workflows.
